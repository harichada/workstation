NAME
    exit - Exit the shell.

SYNOPSIS
    exit [n]

DESCRIPTION
    Exit the shell.
    
    Exits the shell with a status of N.  If N is omitted, the exit status
    is that of the last command executed.

SEE ALSO
    bash(1)
